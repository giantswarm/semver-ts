# semver-ts

A TypeScript library that mimics [semantic versioning](https://semver.org/) comparison and constraints parsing behavior of the [Masterminds/semver](https://github.com/Masterminds/semver) Go library.

Written with the goal to emulate upgrade detection for [Flux CD OCIRepository](https://fluxcd.io/flux/components/source/ocirepositories/) resources in a TypeScritp/NodeJS context (a Backstage portal).
